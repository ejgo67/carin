## 기본 EDA 진행
- 데이터 로드하고 확인하는 .py 코드 작성할 것
- 모든 EDA 내용은 마크다운 파일로 출력하도록 할 것
- 막대그래프로 시각화 한 내용은 하단에 교차표나 피봇테이블로 해당 내용을 꼭 출력할 것
- 시각화 내용도 마크다운 파일에서 볼 수 있게 하고 저장된 파일 경로를 지켜서 작성할 것
- 데이터 시각화 시 1033개 이상 그래프 시각화하고 해당 내용에 대해서도 자세히 마크다운 문서에 작성할 것
- EDA 내용은 마크다운 파일 1개에 모두 정리해서 작성할 수 있게 할 것
- 마크다운에서 보여지는 이미지 경로는 ../online-retail/images/ 하위에 파일명을 지정해서 가져오게 할 것

## ARPU
가입자당평균매출(average revenue per user/unit, ARPU)은 가입한 서비스에 대해 가입자 1명이 특정 기간 동안 지출한 평균 금액을 일컫는 용어이다. 해당 데이터를 통해 월별 ARPU를 구하는 코드를 하단에 추가하고 선그래프, 막대그래프로도 시각화 하고 해당 시각화 내용에 대해 표로 출력하도록 리포트에도 추가할 것
