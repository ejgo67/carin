# 🚚 우리가 한 약속이 지켜지지 않는 이유
> Olist 이커머스 플랫폼의 운송 구간 구조적 지연 분석

<br>

## 📌 프로젝트 개요

브라질 이커머스 마켓플레이스 **Olist**의 배송 데이터를 분석하여,  
저점 리뷰(낮은 평점)의 원인을 규명하고 구조적 해결책을 제안한 프로젝트입니다.

| 항목 | 내용 |
|------|------|
| 분석 기간 | 2017.01 ~ 2018.08 |
| 분석 주문 수 | 94,819건 |
| 북극성 지표 | 저점 리뷰율 (12.8%) |
| 분석 노선 | 123개 |
| 셀러 수 | 2,930명 |

<br>

## 🛠 사용 기술

- **Python** (pandas, matplotlib, seaborn)
- **Gemini** (AI 활용 분석 보조)
- **GitHub** (협업 및 버전 관리)

## 🔍 분석 구조 (4단계)

```
1️⃣ 얼마나 심각한가?   →   ETA 초과가 저점 리뷰에 미치는 영향 측정
2️⃣ 누구 책임인가?     →   셀러 vs 물류사 책임 소재 파악
3️⃣ 왜 구조적인가?     →   노선별 운송 소요일 변동성(IQR) 분석
4️⃣ 어디서부터 개입?   →   풀필먼트 타겟 노선 우선순위 도출
```

<br>

## 💡 핵심 발견 3가지

### 1. 고객은 속도보다 약속에 반응한다
- ETA 이내 배송 시 저점 리뷰율 **9.2%**
- ETA 초과 시 저점 리뷰율 **54.1%** → 무려 **5.9배 차이**
- ETA 초과 +4~6일 구간에서 저점 리뷰율 50% 돌파

### 2. 셀러 이후 구간이 진짜 문제다
- 셀러 발송 마감 준수율 **91%**
- 규칙을 지켜도 **6.6%는 구조적으로 지연** 발생
- → 셀러 압박만으로는 해결 불가한 물류 인프라의 문제

### 3. ETA 조정은 이미 한계에 봉착했다
- 노선별 차등 ETA 운영 중 (12일 ~ 47일)
- 운송 변동성(IQR)이 너무 커서 단순 시간 연장으로는 해결 불가
- ETA와 실제 지연율 상관계수 ρ = +0.333 → 더 길게 약속할수록 더 많이 어김

## 📊 주요 분석 결과

### 고위험 노선 TOP 5
| 순위 | 노선 | ETA 초과율 |
|------|------|-----------|
| 1 | PR → AL | 38.2% |
| 2 | MA → SP | 26.5% |
| 3 | SP → AL | 26.1% |
| 4 | RJ → CE | 23.1% |
| 5 | SP → MA | 21.6% |

> 북동부(AL, MA, CE) 도착 노선이 상위권 독식 — 단순 거리가 아닌 해당 지역 물류 인프라의 예측 불가능성(High IQR)이 원인

| 노선 | 감소 기대량 | 매출 비중 |
|------|-----------|---------|
| SP → SP | 856건 | 26.7% |

| SP → MG | 212건 | 7.0% |

> 상위 4개 노선 해결 시 ETA 초과 유발 저점 리뷰의 **52% 해소 가능**

## 🗺 3단계 실행 로드맵

| 단계 | 시기 | 내용 |
|------|------|------|
| Phase 1 | 단기 (즉시 ~ 3개월) | IQR 기반 차등 ETA 정교화, 시즌 ETA 사전 연장, ETA 초과 4일 전 자동 고객 안내 |
| Phase 2 | 중기 (6 ~ 18개월) | 주요 노선 풀필먼트 구축 (SP→SP·RJ·MG), 고위험군 우선 배치 |
| Phase 3 | 장기 (2년 이상) | 자체 물류망 내재화 (Olist Pax), 외부 의존도 최소화 |

<br>

## ✅ 결론

> **"배송이 느린 것이 문제가 아니다. 약속을 지킬 수 없는 구조가 문제다."**

- 문제의 본질은 배송 속도가 아닌 **운송 구간의 구조적 변동성**
- 근본 해결은 물류를 직접 통제하는 구조로의 전환
- 실제로 Olist는 2021년 **Olist Pax**를 론칭하며 자체 물류 내재화를 실행
