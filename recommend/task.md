## 유사도 분석 및 추천 시스템
- 유사도 기반으로 검색을 하면 키워드가 일치하지 않더라도 가져옵니다.
- 유사도는 데이터를 숫자로 변환합니다=>유사도를 계산합니다 =>유사도 값이 높은 순으로 가져옵니다.
- df_valid['Description'] 에서 물건의 유니크값을 구합니다. 물건 종류 수가 나옵니다.
- 유니크한 문자를 숫자로 변환합니다.
- 이때 사용하는 기법이 어떤 단어가 있고 해당 단어가 가중치가 얼마인지를 구합니다 => TF-IDF
- 이 숫자를 계산해서 어떤 단어끼리 가깝고 먼지 거리를 구합니다.
- 거리가 가까운 순으로 추천합니다.

### 컨텐츠 기반의 추천시스템의 시작
- df_valid['Description'] 에서 유니크 값을 구하고 TF-IDF로 가중치를 구한뒤 유사도 분석을 진행하고 랜덤하게 5개 정도의 물건을 뽑아서 유사도가 높은 순으로 10개씩 Description 을 유사도와 함께 출력할 것

- df_valid['Description'] 에 있는 유니크한 값들을 텐서플로우 임베딩프로젝터에 띄우기 위해 허깅페이스의 가장 가벼운 모델로 임베딩을 하고 메타정보와 벡터정보에 대한 tsv 파일을 만들것

- df_valid['CustomerID'], df_valid['Description'] 으로 user-item 매트릭스를 만들고 간단한 협업필터링을 구현한뒤 유저 5명을 랜덤하게 찾고, 해당 유저마다 유사도가 높은 사용자를 찾아서 상품을 10개씩 추천하고 상품명과 유사도 수치를 함께 출력할 것
